# KoTerm

# What is KoTerm?
KoTerm is a lightweight REPL terminal built entirely in **Kotlin**.\
This project is developed for learning purpose – 
to deepen my understanding of Kotlin's standard library, string parsing algorithms, and core data structures.\
It simulates the basic behavior of a command-line interface making it a perfect playground for learning.

# How to install KoTerm

Clone this repository:
```bash
git clone https://github.com/kostya05983/KoTerm.git
```
### 1. Via IntelliJ IDEA   
1. Open the project in IntelliJ IDEA
2. Run the `Main.kt` file
3. Enjoy!
